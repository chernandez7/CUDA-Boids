# CUDA-Boids

Senior year Independent Study (COMP 398) reworking [original Bird-oids project](https://github.com/chernandez7/Boids).


### Goals
 - 3D implementation.
 - CUDA for computing in parallel.
 - OpenGL for rendering.
 - Better efficiency.

### To Run
 - Compile
	-`make`
 - Run
	-`./CUDA-Boids`
